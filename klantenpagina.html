de cloud

wat is cloud

Veel mensen zijn onzeker over wat cloud opslag (‘cloud storage’) werkelijk is en hoe het werkt. Om het simpel te zeggen, cloud opslag betekent dat er gegevens worden opgeslagen op een off-site opslag systeem. Het onderhoud van dit opslagsysteem wordt gedaan door een derde partij. Vanwege het feit dat alle besparing plaatsvindt op een externe database is er internet nodig om de overdracht van gegevens van jouw apparaat naar het externe opslagsysteem te volbrengen

voordelen van cloud

De voordelen van cloud servers ten opzichte van lokale servers zijn aanzienlijk. Met cloud opslag loop je niet tegen bekende problemen aan zoals bestanden die worden aangetast doordat de computer crasht, wordt gestolen of andere computer schade, dus het maakt niet uit welke technologische tragedie je zou doorstaan, je bestanden worden opgeslagen met Cloud opslag. Cloud opslag zorgt voor toegang van de opgeslagen gegevens vanaf elke computer, mobiel, tablet of locatie met een internetverbinding. Met cloud opslag kun je gemakkelijk bestanden delen met vrienden, familie of collega’s, om toegang te krijgen tot gegevens moet er simpelweg ingelogd worden of bestanden gesynchroniseerd zijn, in team verband is een cloud oplossing dus ideaal. Een ander voordeel van de cloud is besparingen op onderhoudskosten, omdat het onderhoud wordt uitgevoerd door een derde partij. Met een cloud oplossing kun je eenvoudig de opslag mogelijkheden uitbreiden om behoeften te voldoen, je bent niet gelimiteerd zoals dat met lokale opslag gaat, telkens een grotere hardeschrijf aanschaffen of meer geheugen of een nieuwe usb-stick.

wat is webhosting

Hosting is in feite het huren van een stukje opslag, rekenkracht en geheugen van een server (dat is een hele sterke computer die meestal in een speciaal datacenter staat) bij een webhoster zoals SoHosted. De webhoster zorgt ervoor dat je website en e-mail blijven werken door updates uit te voeren, de server te beveiligingen en de server zo te configureren dat elke website snel draait.

soorten hostingpakketen

Shared webhosting

De meeste voorkomende vorm van hosting is ‘shared webhosting’ ook wel webhosting genoemd. Hiermee huur je een stukje opslag en rekenkracht op een server. De server deel je met andere klanten. Omdat anderen ook gebruik maken van dezelfde server, zijn de maandelijkse kosten ook heel laag. Onze shared webhosting is anders ingericht dan de shared webhosting van de meeste hostingbedrijven. Bij shared webhosting van SoHosted krijg je een afgeschermd stukje van de server (eigen CPU & RAM) zoals bij een VPS, hierdoor hebben andere websites geen invloed op de prestaties van jouw website waardoor de snelheid en stabiliteit toeneemt. Bovendien maakt onze shared webhosting gebruik van 100% SSD opslag waardoor je website razendsnel is. Met alle hostingpakketten is het mogelijk om je website te beveiligen met een SSL certificaat en kun je gebruik maken van Microsoft SQL Server databases en het full trust level voor ASP.NET.

Reseller hosting

Een andere vorm van hosting is reseller hosting. Hiermee huur je een virtuele server (VPS) waarmee je zelf hosting kunt aanbieden aan je eigen klanten. Je kunt dan dus zonder dat je een eigen server hoeft aan te schaffen webhosting aan klanten verkopen.

VPS hosting

Dan is er nog VPS hosting. Met VPS hosting huur je een Virtual Private Server (een eigen virtuele server) die heel snel is, een hoge uptime heeft en schaalbaar is. VPS hosting deel je niet met andere klanten; de opslag, rekenkracht en geheugen van de server zijn geheel voor jou. Ook kan je zelf kiezen welk besturingssysteem je op je VPS wilt draaien en kan je er meerdere websites op laten draaien.
